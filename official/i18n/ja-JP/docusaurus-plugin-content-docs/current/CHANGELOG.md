# 変更ログ

## 1.12.3 (2024-12-13)

- 追加：**AI コンテキスト認識翻訳**により、専門的なコンテンツの翻訳精度を向上させることができます。（Pro会員のみ利用可能）【**設定**】->【**基本設定**】->【**AI コンテキスト認識翻訳を有効にする**】
- 修正：Twitter の複数行翻訳効果における一部のバグを修正
- 修正：リッチテキストによる特定の数式翻訳の問題を修正
- 改善：**x.com** での翻訳時、字幕付きの動画は自動的にバイリンガル字幕に翻訳されます
- 改善：字幕のない動画には翻訳アイコンが表示され、翻訳できない理由が表示されます

## 1.11.7 (2024-11-25)

- 追加：Bing/Google がクメール語（カンボジア語）をサポート
- 追加：未完了の ePub ファイルを再インポート時に、前回の停止位置から翻訳を継続可能に
- 修正：Safari ブラウザでTwitter画像を翻訳できない問題を修正
- 修正：【**マウスホバー翻訳**】機能のオン/オフ切り替え時にショートカットキーが効かなくなる問題を修正
- 改善：Twitter、YouTube の複数行バイリンガル対照翻訳の表示効果を改善
- 改善：バイリンガルモードでは、翻訳品質向上のためデフォルトでリッチテキスト翻訳を無効に
- ~~改善：【**詳細設定**】に【**サイドバーとナビゲーションバーの翻訳**】のカスタマイズオプションを追加~~
- 改善：【**マウスホバー-直接段落を翻訳**】モードで画像を翻訳しないように変更

## 1.11.4 (2024-11-16)

- 修正：バージョン1.11.2の「Twitter翻訳改善」による数式翻訳の問題を修正

## 0.0.34

### 変更点

- 翻訳のトグルに使うデフォルトのホットキーが `alt+A` に変更されました。これは最も便利なキーだからです。

### その他

- 即時翻訳モードを設定できるようになりました。これにより、ウェブページを可能な限り迅速に翻訳できます。
- 翻訳が必要なページエリアを設定できるようになりました。これにより、より多くのエリアを翻訳できます。
- 即時に翻訳する最初のテキスト数を設定できます。
- 翻訳を変更するときに重複して翻訳される問題を修正しました。
- より良いポップアップUI
